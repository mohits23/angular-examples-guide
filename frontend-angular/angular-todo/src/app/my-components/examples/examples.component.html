// "Pipes" are used to manipulate any data before it displayed in the template.
// "Pipes" act like filter for data to be manipulated in a specified format ...
// or in some particular mannner or just replace the data with other data ...
// or append something to the original data.

// The "Pipe" only changes the value of any class "property"
// only for the View(Template).
// Pipe does not original value of the class "property" remains unchanged.

// Denoted by: Pipe Symbol ("|")
// Pipes in Angular act just like "Template Filter" in 'Django'.
// Many useful Pipes have been pre-defined ....
// but Custom pipes can also be made as required.

// In-build "Pipes" in Angular.

{{ "angular framework" | titlecase }} (TitleCase)
